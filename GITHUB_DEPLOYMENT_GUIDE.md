1. **Check repository** is Public (required for free GitHub Pages)
2. **Wait 5-10 minutes** - deployment takes time
3. **Verify file names** exactly match (case-sensitive)

1. **Use Safari** (required for PWA installation)

4. **Restart Safari** if needed

### **Scrolling Not Working on iPhone?**
1. **Use position test page** first: `iphone_position_test.html`
2. **Try different script** (longer text)
3. **Check iOS version** (need iOS 12+)
4. **Clear cache** and reload
